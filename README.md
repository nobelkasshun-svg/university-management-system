# university-management-system

OOP Features



\- Add and manage students and teachers with input validation

\- Enroll students in courses

\- Search students by name, ID, major, or enrolled course

\- View all students and teachers

\- Custom exceptions for invalid input and duplicate/missing records

\- Department and schedule management models



\## How to Run



1\. Make sure you have Java installed (JDK 8 or higher)

2\. Clone the repository:

```bash

&#x20;  git clone https://github.com/nobelkasshun-svg/university-management-system

```

3\. Navigate to the `src` folder and compile:

```bash

&#x20;  cd src

&#x20;  javac main/Main.java

```

4\. Run:

```bash

&#x20;  java main.Main

```



\## Validation Rules



| Field | Rule |

|-------|------|

| Name  | Cannot be null or empty; letters and spaces only |

| Age   | Must be between 1 and 120 |
